퀘스트 구현 할때

퀘스트 갱신 조건이

몬스터 처치 시 (종족 별 or 모든종족) 이러할때

던전에서 몬스터 킬 할때마다 Player의 퀘스트 리스트에서

반복문으로 조건검사를 하는데 이 방식이 뭔가 마음에 안들어서

고민을 해봤는데 딱히 다른 방법이 떠오르지 않았음.

다른 게임에서 수락 가능한 퀘스트가 제한되어 있는 경우가 있던걸 기억하는데

이런 조건이 이와 비슷한 이유로 인해 생긴건지 궁금함.
